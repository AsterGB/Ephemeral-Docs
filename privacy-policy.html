Ephemeral - Privacy Policy

Ephemeral ("the bot") aims to be a general purpose bot that currently covers leveling and moderation. We aim to maintain transparency in what is and is't stored.

Data Storage

The bot stores the following data:

  • Message Content*

    • Message Content may be stored in server-logs if enabled upon edit or deletion if the message is cached.

    Message Content may be temporarily stored for reminders, but is lost upon the bot powering off.

    Message Content is stored in assocation with infractions. This will be deleted upon deletion of the infraction.

    Message Content is not stored otherwise.

  • User Profile*

    • Information of a user's profile is not exactly stored.

    When a user updates their profile, the bot may pick up on the change and forward a notice to a logging channel in any enabled servers.

    The bot may also fetch the user's name and avatar icon for bot-related functions such as displaying a rankcard.

  • Guild Profile*

    • Information of a guild's profile is not exactly stored.

    When someone updates a guild's profile, the bot may pick up on the change and forward a notice to a logging channel if enabled in the server.

    This includes changes to channels and roles.

  • Bot-Related User and Server Information

    • The bot stores data for information that can be changed per-server or requires per-server configuration.

    This is not temporary data.

    This information may be for leveling, moderation, or any other feature supported by the bot that may require user and server information to be stored.

    Third Parties

    Data is not sold to any party.

    Data Deletion

    Please contact astergb regarding data deletion.
